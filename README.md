# TradePulse 📟

**Adopt Me! Trades Value Calculator** — built like the
community tools (Elvebredd, AdoptMeValues, etc.), to guide you through all your trades

Compare two trade offers side-by-side, toggle **Neon / Mega Neon / Fly /
Ride** on any pet, and watch the live "pulse" verdict tell you who's winning.

## What's included

- A live "pulse" verdict bar that shows whether your offer or their
  offer is ahead, and by how much. **Fair trade only triggers when
  both sides are worth the exact same amount** — anything else shows
  as winning or losing, even by a tiny margin.
